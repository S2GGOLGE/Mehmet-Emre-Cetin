Hakkımda

Ben güvenli, yüksek performanslı ve ölçeklenebilir yazılım mimarileri inşa etmeye odaklanmış bir Backend & Full-Stack Developer'ım. Modern web servisleri, veritabanı tasarımları ve karmaşık sistem entegrasyonları üzerinde çalışıyorum.

ASP.NET Core, C#, SQL Server ve JWT tabanlı yetkilendirme altyapılarında derin deneyime sahibim. Ayrıca Python ve Yapay Zekâ teknolojilerini sistemlere entegre ederek akıllı akışlar ve otomasyon çözümleri üretiyorum.

Teorinin ötesinde, gerçek dünya problemlerini çözen uçtan uca sistemler geliştiriyorum. Şu anda akıllı ev ve otomasyon ekosistemini kapsayan HomeOS projesi üzerinde çalışmaktayım.

Öne Çıkan Proje

★ AMBAR PROJESİ

HomeOS Ekosistemi

Kapsamlı Akıllı Ev, Otomasyon ve Yapay Zekâ Asistan Sistemi

Proje Özeti

HomeOS, uçtan uca backend yetkinliklerini sergileyen kapsamlı bir akıllı ev ekosistemidir. ASP.NET Core, SQL Server ve Python Yapay Zekâ entegrasyonu ile inşa edilmiş olup; gerçek zamanlı sistem tasarımı, güvenlik uygulamaları ve yüksek performans sunar.

Teknoloji Yığını

ASP.NET Core C# SQL Server JWT Auth Python FastAPI WebSocket REST API

Öne Çıkan Özellikler

  • Backend API: JWT doğrulama destekli ASP.NET Core REST mimarisi
  • Veritabanı: Optimize edilmiş SQL Server ilişkisel veritabanı
  • Yapay Zekâ Motoru: WebSocket entegrasyonlu Python FastAPI servisi
  • Cihaz Yönetimi: Gerçek zamanlı IoT cihaz kontrolü ve senkronizasyonu
  • Otomasyon: Kural tabanlı (If-This-Then-That) otomasyon motoru
